# ❌⭕ Tic-Tac-Toe

A modern and responsive Tic-Tac-Toe game built with **React** and **Vite**. Challenge the CPU, keep track of your scores, invite friends to play, and enjoy motivational quotes while playing.

---

## 📦 Deployment

The application is deployed on **Vercel**.

Live URL:

https://tic-tac-toe-git-main-shreya-ranjans-projects.vercel.app/

---


## 🚀 Features

- 🎮 Play Tic-Tac-Toe against the CPU
- ❌ Choose your preferred symbol (X or O)
- 🏆 Automatic score tracking
- 💾 Scores are saved using Local Storage
- 📋 Copy game link using the "Invite your Friend" button
- 💬 Displays motivational quotes fetched from the Advice Slip API
- 📱 Responsive user interface
- ⚡ Built with Vite for fast performance
- 🔔 Toast notifications for user interactions

> **Note:** Multiplayer (VS Human) is currently marked as **Coming Soon**.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Shreyara18/tic-tac-toe.git
```

### Navigate to the project

```bash
cd tic-tac-toe
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

---

## 🎮 Gameplay

1. Select your preferred symbol (**X** or **O**).
2. Start a new game against the CPU.
3. Take turns placing your moves.
4. Scores are automatically updated and stored locally.
5. Use **Invite your Friend** to copy the game URL.

---

## 💾 Persistent Storage

The application stores:

- Selected player
- Game state
- Scoreboard

using the browser's **Local Storage**, allowing your progress to persist even after refreshing the page.

---

## 💬 Advice API

The application periodically fetches motivational advice from the **Advice Slip API** to display alongside the game.

If the API is unavailable, a default quote is shown.
